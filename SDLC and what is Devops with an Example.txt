Devops:
--------

DevOps is a culture and is a process that removes the gap between development, testing, and operations teams. 
Instead of working separately like in older models, all three teams work together from the beginning. 
DevOps focuses on collaboration and automation using CI/CD pipelines to build, test, and deploy code quickly. 
Testers perform continuous testing from the start and the operations team gets involved early to set up infrastructure and monitoring. 
This approach enables faster deployments, early bug detection, continuous delivery of features, better product quality, 
and strong teamwork between Dev, Test, and Ops.

Earlier in Waterfall, developers wrote code for months, testers waited till the end, and ops deployed manually ‚Äî so everything was delayed. 
Agile solved the dev and tester delay by delivering in small sprints, but ops delay was still there because deployment was manual. 
DevOps fixed that by automating build, test, and deployment, and by making dev, test, and ops work together ‚Äî so software can go live the same day.

What is DevSecOps?

*)It is an improved version of DevOps.
*)In DevOps, security was mostly checked at the end, which caused delays if issues were found late.
*)DevSecOps brings the Security team into the DevOps process from the start.
*)Security is built into every stage ‚Äî from coding to deployment
*)Developers, testers, operations, and security all work together from the beginning
*)Automated security checks are added in CI/CD pipelines.
*)Fix security issues early before going to production.
